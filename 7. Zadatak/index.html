Web dizajn

HTML

HTML je skraćenica od engleskih reči HyperText Markup Languagešto u prevodu znači jezik za označavanje hipertekstova, odnosno tekstova sa linkovima, slikama, audio i video zapisima.

CSS

CSS (engl. Cascading Style Sheets) je jezik formatiranja pomoću kog se definiše izgled elemenata web stranice.
