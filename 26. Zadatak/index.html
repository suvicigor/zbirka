Web dizajn

Web dizajn je proces prikupljanja ideja i njihovog estetskog uredjivanja i rasporedjivanja radi postizanja odredjenog cilja, a to je izrada web sajta.

Web dizajn podrazumeva upotrebu raznih vestina i disciplina koje se koriste za planiranje i izradu web sajtova.

Responsive web dizajn

Responsive web dizajn podrazumeva da je web sajt vidljiv na svim uredjajima, a ne samo na desktop racunarima. Mobilni telefoni, tableti, laptopovi - svi ovi uredjaji moraju biti uzeti u obzir priliko dizajna web sajta.
